# 常见问题一：我想做 xxx 赛道，应该怎么做？

这是一个错误的问题。

如果你不知道怎么做这个赛道，你就不应该选择这个赛道。

这个赛道能不能赚钱不重要，你自己能不能赚钱，很重要。



所以，去多看看各种各样的生意，不要看泛泛而谈的行业分析。

媒体上天天说给欧美人卖水晶赚钱、给北上广卖情趣用品赚钱，这和你又有什么关系呢？



不要尝试去回答一个错误的问题，浪费时间。



# 常见问题二：我今天发了 2 篇小红书，被限流了怎么办

1. 你有没有找到一个赚钱的对标账号？这决定了你现在做的内容有没有意义。如果你的对标账号就不赚钱，那你就算搞到了流量也是浪费时间

2. 你在发帖之前，有没有解决网络问题、账号问题、设备问题，能不能确保是内容问题，而不是其他问题

3. 就算是内容有问题，算法本身有巨大的随机性。你让小红书内部的运营总监来发帖，也不可能一天就爆火吧？



# 常见问题三：我现在遇到了 xxx 问题，觉得很有挫败感

要建立正确的负反馈体系，搞清楚失败归因。



比如：我在抖音（平台）通过做创始人 IP（打法）做宠物领域（赛道）的鲜食（细分品类），最后发现能引流（阶段性成功），但是不好变现（利润款、产品结构）

**二维的失败归因是：**&#x6211;失败的地方在于产品结构有问题，但是赛道、打法、细分品类，都是没问题的。
这是一个递进的结构，要看一下需要回滚到哪一层，再次出发。
摆事实、讲道理、逻辑推理，不吐槽、不抱怨、没情绪。

**而一维的失败归因是：**“抖音红利期已经过了，根本搞不了”，然后晚上喝顿大酒，留下两行热泪



**如果你搞清楚了，你到底失败在哪个环节、哪个层级，哪个具体的问题，你需要什么样的资源或者帮助，你可能就会忙于解决问题，而没时间内耗。**

对于失败一头雾水，不知道如何解决，才会内耗。



# 常见问题四：我看见有人做 xxx 搞到了流量，我能不能学她，然后卖 xx 产品赚钱？



新手的赚钱路径是：

1. 找一个赚钱的对标

2. 研究清楚这个对标

3. 模仿这个对标

4. 赚钱，收款



如果你看见 xxx 搞流量但是没变现，那么第一个步骤就错了。

你说你想试一下行不行，我觉得没问题。

但是万一失败了，你是否可以做到不要心灰意冷？

如果能做到，我建议你试试。如果做不到，那就按照高成功率的方法做事，也就是：



1. 找一个赚钱的对标

2. 研究清楚这个对标

3. 模仿这个对标

4. 赚钱，收款



不是说不能搞创新赚钱，而是人类天然容易产生挫败感。

失败了三五次，你很可能就干不下去，想回去打工了。

用高成功率的方法做事，可以让你少一点挫败，坚持得久一点。

坚持得足够久，就能看到成功，拿到利润。
