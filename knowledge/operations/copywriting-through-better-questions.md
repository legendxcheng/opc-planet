# 好文案来自更好的问题

## Summary

一人公司文案的质量，常常取决于写作前提出的问题质量。把“我适不适合做”“这个赛道能不能做”这类主观问题，改写成关于人群、场景、条件、成本、失败原因的客观问题，能显著提高文案判断和客户研究的质量。

## Key Points

- 问错人会得到无效建议；应优先问最近做过、仍在场内、且有具体经验的人。
- 主观问题容易得到口号，客观问题更容易得到可执行信息。
- 文案前的问题应围绕：谁、在什么场景、为什么痛、已经试过什么、愿意付出什么成本。
- 一人公司不要问“有没有机会”，而要问“机会成立需要哪些前提”。
- 客户访谈、竞品拆解和内容选题都应先改写问题，再开始写文案。

## Details

### 问题改写表

| 低质量问题 | 高质量问题 |
| --- | --- |
| 现在做自媒体还有机会吗？ | 零经验小白做成自媒体通常需要哪些条件、周期和失败原因？ |
| 我适合做这个赛道吗？ | 这个赛道赚钱的前提条件是什么，我缺哪几项？ |
| 这个产品能卖吗？ | 哪类人已经为类似问题付费，他们为什么买？ |
| 怎么快速赚钱？ | 哪些赚钱方式前期投入最低、失败成本可控、反馈周期最短？ |
| 我的文案好不好？ | 这段文案是否清楚说明对象、痛点、承诺、证据和下一步？ |

### 用在文案流程中

写任何一人公司文案前，先回答五个问题：

1. 这段文案写给谁？
2. 他现在遇到的具体场景是什么？
3. 他过去为什么没解决？
4. 我能提供什么更低成本或更高确定性的方案？
5. 我不承诺什么？
